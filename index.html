Welcome to the Student Management System

Introduction

This system allows you to manage students, courses, and skills efficiently. Use the navigation links above to add new students, courses, or skills to the system.




System Overview

Our student management system provides the following features:

  • Add and manage students: Enter student details and keep track of their courses and skills.
  • Add and manage courses: Create and organize courses that students can enroll in.
  • Add and manage skills: Track skills that students can acquire or improve.
